# 빌드 파이프라인

## 전체 흐름

```
원본 ISO
  │ 03_bnd_toc.py / 25_build_db.py          … BND TOC·텍스트 DB(db.json) 구축
  │ 51/86/87/99/100/101 …                   … 폰트 스캔·안전 슬롯·커버리지 분석
  │ 103/104, 122/123 …                      … 번역 배치 생성 → (번역) → 하드검증 병합(tl/*.json)
  ▼
build.py          번역 반영 + kr_map_shell 생성 + 폰트 13종/메인폰트 한글 주입  → ACE3_KR.iso
  ▼
222_patch_caption_batang.py    캡션폰트 8종: 원본 텍스처 복원 → 바탕 비트맵 재주입
                               + 라디오 텍스트 재인코딩(공백/부호/치환)          → ACE3_KR_caption_batang.iso
  ▼
223_shell_batang_unify.py      셸 6종 + 경보 폰트: 동일 스타일 재주입            → ACE3_KR_caption_full.iso
  ▼
xdelta3 -e -f -s 원본.iso ACE3_KR_caption_full.iso 패치.xdelta
  (그리고 -d로 재적용 → SHA256 라운드트립 검증)
```

## 번역 병합 하드 게이트

`123_merge_story.py` 등 병합 도구는 다음 불일치 시 병합을 거부한다:
- 앵글 태그 멀티셋 불일치 (`<ab(800)>` 등 개수·철자)
- 개행 수 불일치
- `⟦XXXX⟧`/`%d` 서식 소실

## 오버플로 처리 (build.py)

컨테이너 힙이 원본 크기를 넘으면, **가장 많이 늘어난 문자열부터 하나씩 원문으로
되돌리며 재시도**하는 점진적 부분 폴백을 쓴다(전체 롤백 금지 — 92바이트 초과로
315개 번역이 날아가는 사고 방지).

## 인게임 검증 (emu.py)

- PCSX2 창을 pid+프로세스 경로로 특정(제목만으로는 브라우저 오탐).
- 전역 keybd_event 입력은 **포커스 실패 시 ALT가 메뉴바를 활성화해 VM이 꺼질 수 있다**
  — 렌더 영역을 마우스 클릭해 포커스 확보 후 키 입력.
- 키맵: `v`=START, `x`=○(결정), `z`=×(취소). Enter는 PCSX2 pause 토글과 겹치므로 금지.
- 두 인스턴스 동시 실행 금지(단일 인스턴스 충돌로 VM 종료).
- 깊은 진단은 GS 덤프 역추적(엔진 렌더러 제약 문서 참조).

## 배포물 검증

```
xdelta3 -d -f -s 원본.iso 패치.xdelta 재적용.iso
sha256(재적용.iso) == sha256(빌드결과.iso)   # 필수 통과
```
